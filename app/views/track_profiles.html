Track Profiles

Track/Distance profiles are available for all race distances at Gulfstream Park. Some distances race over both turf and dirt, where this is the case, dirt is covered first, turf second, under their own headings. Subjects for analysis include:

  • Draw
  • Pace (via Finishing Speed %'s)
  • Distance travelled (via Trakus data)

The content behind each tab, is dynamically created using the rmarkdown package, which means the document stays consistent, but any analysis (code output) is updated to include the latest data from races - the date shows when the content was last updated.
