Track Profiles

Track/Distance profiles are available for all race distances at Gulfstream Park. Some distances race over both turf and dirt, where this is the case, dirt is covered first, turf second, under their own headings. Subjects for analysis include:

  • Draw
  • Pace (via Finishing Speed %'s)
  • Distance travelled (via Trakus data)

The content behind each tab is created in R using the rmarkdown package. This means content is dynamically created and any analysis (code output, plots) is updated to include the latest data from races. The Last Updated text is part of the dynamically created document, and will be the date on which the document/report was created.

Over time I hope to write more detailed, specific, analysis for each distance, moving away from a single template for all distances, but this is a start. The single template for all distances means language in profiles is quite repetitive, however the data behind the profiles, fuelling the plots and bold sections will be unique to each distance.
