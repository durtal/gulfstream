About

This web app contains ratings for individual horses who have raced at Gulfstream Park from December 2012 onwards. Ratings have been created using data collected from a variety of sources, with the bulk of the work being done using the R programming language and a package I created called RcappeR, which is available on github for others to use. I hope that ratings found here, and posts that will walk through the handicapping process will encourage others to use the package.

These ratings would not be possible without a number of open source tools, these include:

  • R - programming language
  • Git - version control, Github also hosts this app
  • MongoDB - database
  • AngularJS - web app framework
  • Yeoman - web app scaffolding tool, uses 2 more open source tools, Grunt and Bower
