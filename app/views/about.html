About

This web app contains ratings for individual horses who have raced at Gulfstream Park from December 2012 onwards.

Ratings have been created using data collected from a variety of sources. With the data collected, the bulk of the work being done using the R programming language and a package I created called RcappeR, which will eventually be available on github for others to use. I hope that ratings found here, and posts that will walk through my handicapping process will encourage others to use the package, and to share with other race fans.

I have taken advantage of open source software in the creation, storage, saving and presentation of the ratings, and other content found in this web app. The primary software I have used include:

  • R - programming language
  • Git - version control, Github also hosts this app
  • MongoDB - database
  • AngularJS - web app framework
  • Yeoman - web app scaffolding tool, uses 2 more open source tools, GruntJS and Bower

It was June 2013 that I started to learn R on an my wordpress blog - I didn't know any HTML, or CSS at that point either. At the time I had no idea it would open the door to the other technologies I've since attempted to learn and employ.
