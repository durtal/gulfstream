About

This web app contains ratings for individual horses who have raced at Gulfstream Park from December 2012 onwards.

Ratings have been created using data collected from a variety of sources. With the data collected, the bulk of the work has been carried out using the R programming language and a package I created called RcappeR, which will eventually be available on github for others to use. I hope that ratings found here, and posts that will walk through my handicapping process will encourage others to use the package, offer suggestions, improvements, alternatives, and to share with other race fans.

It was June 2013 that I started to learn R on my old wordpress blog, I also knew very little HTML and CSS, just enough to highlight some clumsy R code. At the time I had no idea I would end up with a web app like this (which is still very much a work in progress). I have taken advantage of a lot of open source software in the creation, storage, saving, building, and presentation of the ratings/analysis found in these pages. The primary software I have used includes:

  • R - programming language
  • Git - version control, Github also hosts this app
  • MongoDB - database
  • AngularJS - web app framework
  • Yeoman - web app scaffolding tool, uses 2 more open source tools, GruntJS and Bower
